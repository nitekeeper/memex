# lessons/

Lesson capture for Memex. Mirrors Skill Atelier's `lessons/` structure.

- `inbox/` — AI-captured lessons awaiting review
- `feedback/` — user-direct feedback (higher priority than inbox)
- `promoted/` — lessons approved and promoted into methodology

## Cadence

- Capture: after any non-trivial session
- Review: at every session close (same cadence as Skill Atelier)
