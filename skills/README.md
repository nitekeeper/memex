# skills/

The skill files that make up the Memex product. Populated during the build phase.

Anticipated skills:
- `capture/` — write a project-wiki page from a session
- `sync/` — staleness detection via `synced-at-commit` + `describes-files`
- `search/` — FTS5-powered search across project wiki pages
- `capture-lesson/` — lesson capture loop
- `review-lessons/` — lesson review + promotion
- `propose-wiki-entry/` — propose a wiki entry during a session
- `review-wiki/` — quarterly wiki curation
